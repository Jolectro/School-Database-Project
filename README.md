#School Database Management System (Excel)
##Overview
This is a comprehensive Excel-based School Database Management System designed to manage student, teacher, class, and school administrative data efficiently. The system includes automated calculations, dashboards, and inventory tracking — making it suitable for portfolio projects, portfolio showcases, or educational purposes.

##Features
###1. Student Management
Stores student details (ID, name, class, etc.)
Tracks subjects, test/exam scores, total scores, and grades
Automated Student Result Profile using XLOOKUP

###2. Teachers & Classes
Stores teacher information
Links students to classes
Calculates total students per class

###3. Results & Performance Dashboard
Dynamic per-student performance chart
Grade distribution 
Automatically updates based on student selection

###4. Inventory Management
Tracks items (furniture, stationery, ICT, lab, sports)
Calculates stock balance automatically
Highlights items that need reorder

###5. Other Modules
Attendance tracking
Fees management
Club activities
Timetable management
Discipline records

##Technology
Excel with formulas: XLOOKUP, COUNTIF, COUNTIFS, SUM, AVERAGE, IF
Formula-automated tables 
Locked formula columns for data protection

##How to Use
Open the Excel workbook.
Enter student IDs in Results to view results.
Dashboards update automatically based on input.
Use the Class, Inventory, and Fees sheets to track school operations.

##File Structure
Students — student details
Teachers — teacher records
Subjects — subject listings
Class — class information
Results — all student scores
Attendance — attendance tracking
Fees — fees tracking and calculations
Inventory — school assets tracking
Discipline — behavior tracking
Club Activities — extra-curricular records
Timetable — class schedule
Dashboards — Results dashboard
